# Cultural-Heritage-Image-Classification

## 설치

```
$ pip install flask
$ pip install google-cloud-vision
$ pip install keras
$ pip install tensorflow
```

## 실행

-   vision ai api key 삽입
-   app.py의 client에 api key 작성</br>
